# Imágenes del proyecto

Coloca aquí las imágenes estáticas. Se sirven desde la raíz del sitio.

## Login
Guarda la foto del login en esta carpeta con el nombre exacto:

    login-cumbre.jpg

Quedará referenciada en el código como `/images/login-cumbre.jpg`.
Si la guardas en otro formato (.png, .webp), avísame para ajustar la extensión.
